BUILT IN PYTHON FUNCTIONS
SYNTAX DESCRIPTION
abs() Returns the absolute value of a number
all() Returns true if all items in an iterable object are true
ascii() Returns a readable version of an object.
bin() Returns the binary version of a number
bool() Returns the boolean value of the specified object
bytearray() Returns an array of bytes
bytes() Returns a bytes object
callable() Returns True if the specified object is callable, otherwise False
chr() Returns a character from the specified Unicode code.
classmethod() Converts a method into a class method
dict() Returns a dictionary (Array)
eval() Evaluates and executes an expression
filter() Use a filter function to exclude items in an iterable object
float() Returns a floating point number
hex() Converts a number into a hexadecimal value
input() Allowing user input
int() Returns an integer number
len() Returns the length of an object
map() Returns the specified iterator with the specified function applied to each item
max() Returns the largest item in an iterable
min() Returns the smallest item in an iterable
open() Opens a file and returns a file object
pow() Returns the value of x to the power y
print() Prints to the standard output device
repr() Returns a readable version of an object
round() Rounds the numbers
setattr() Sets an attribute of an object
slice() Returns a slice object
sorted() Returns the sorted list
str() Returns a string object
sum() Sums the items of an iterator
tuple() Returns a tuple
capatalize() Converts the first letter into uppercase
casefold() Converts string into lowercase
isalnum() Returns true if all characters in a string are alphanumeric
isascii() Returns true if if all characters in the string are ascii characters
islower() Returns true if all the letters in the string are lowercase
isupper() Returns true if all the letters in the string are uppercase
