LIBRARIES AND BUILT-IN MATH FUNCTIONS OF C++ PROGRAMMING
SYNTAX DESCRIPTION
c++ cos() Returns cosine of the argument
c++ sin() Returns sine of the argument
c++ asin() Returns inverse sine a number
c++ acos() Returns inverse cosine a number
c++ atan() Returns inverse tangent a number
c++ tan() Returns tangent of the argument
c++ atan2() Returns Inverse Tangent of a Coordinate
c++ sinh() Returns hyperbolic sine of an angle
c++ tanh() Returns hyperbolic tan of an angle
c++ cosh() Returns hyperbolic cosine of an angle
c++ ceil() Return ceiling value of number
c++ floor() Returns floor value of decimal number
c++ fmod() Computes floating point remainder of division
c++ acosh() Returns arc hyperbolic cosine of a number
c++ asinh() Returns arc hyperbolic sine of a number
c++ atanh() Returns arc hyperbolic tangent of a number
c++ log() Returns Natural Logarithm of a Number
c++ lround() Returns the long int value nearest to the argument
c++ exp() Returns exponential (e) raised to a number
iostream iostreams are the formatting part of a two-part (input or output) system. The other part of the system is made up of streambuf s, which deal in input or output of unformatted streams of characters.
using namespace std Meaning that we can use names for objects and variables from the standard library.
cmath Contains various math library functions.
iomanip Contains stream manipulator functions that allow us to format the stream of data.
cstdlib Contains various functions related to conversion between text and numbers, memory allocation, random numbers, and other utility functions.
ctime Contains function prototypes related to date and time manipulations in C++.
cctype Includes function prototypes that test the type of characters (digit, punctuation, etc.). It also has prototypes that are used to convert from uppercase to lowercase and another way around.
cstring Includes function prototypes for C-style string-processing functions.
climits Contains the integral size limits of the system.
cassert Contains macros and variables for adding diagnostics that help us in program debugging.
cstdio This header contains function prototypes for the C-style standard input/output library functions which we included initially in stdio.h
cfloat Contains the size limits for floating-point numbers on the system.
memory Used by C++ standard library to allocate memory.
sstream Functions that read input from strings in memory and output to strings in memory require functions prototypes from sstream header to implement the functionality.
iterator Function prototypes and classes in this header are used by Standard Template Library to traverse through or iterate through the data inside containers.
algorithm Methods that act on STL container data are included in this header algorithm
locale To process data in the original natural form for different languages or locales (currencies, character presentation, etc.), the locale header definitions are used.
limits This header defines the data type limit for Numbers on each platform.
utility This header contains utility functions and classes used by the Standard C++ library.
